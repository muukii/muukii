*All OSS projects are used in Pairs and have proven track records in real-world use cases*

Key principles:
- Well-decomposed components by necessity
- Solving problems through abstract thinking

**Featured Projects** (selected from many open source contributions):

#### [Verge](https://github.com/VergeGroup/swift-verge)

State management framework optimized for the UIKit era

- Implementation based on the store-pattern concept underlying Flux
- Multi-threading support

#### [swift-state-graph](https://github.com/VergeGroup/swift-state-graph)

State management for modern SwiftUI-era applications

- Developed to support UIKit + SwiftUI environments
- Original idea inspired by Recoil for React

#### [FluidInterfaceKit](https://github.com/FluidGroup/FluidInterfaceKit)

Custom navigation and presentation framework

- Implemented before iOS 18's ZoomTransition existed
- Built NavigationController and Modal-Presentation mechanisms from scratch

#### [Brightroom](https://github.com/FluidGroup/Brightroom)

Image editing library using CoreImage

- Image processing layer and UI layer are separated, allowing product-specific UI development
- While there's still room for improvement, the implementation for rotating and cropping images was the most challenging work
